## Installation

Run npm -i to install dependencies

## Usage

You can query "Drinks" resources as shown below:

POST '/api/drink' [name, flavor, isAlcoholic] - to post a new drink
GET '/api/drink/:id' - to get a specific drink

The API handles for errors so it should prevent a user from making too many mistakes.
